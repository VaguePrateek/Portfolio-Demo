# Portfolio Demo

A responsive personal portfolio website showcasing professional work, background, and contact information for **Prateek Dewangan**.

## Overview

This project is a modern, clean portfolio website built with HTML and CSS. It features a multi-page layout with smooth navigation, responsive design, and social media integration. The site is optimized for both desktop and mobile viewing experiences.

## Features

- **Responsive Design** - Mobile-friendly layout that adapts to all screen sizes
- **Multiple Pages** - Home, About Me, Portfolio, and Contact pages
- **Social Media Integration** - Quick links to Instagram, LinkedIn, and GitHub
- **Modern Styling** - Custom CSS with Google Fonts (Josefin Sans & Playwrite VN)
- **Navigation Menu** - Responsive navbar with mobile sidebar support
- **Font Awesome Icons** - Beautiful social media and UI icons

## Project Structure

```
Portfolio-Demo/
├── Home.html              # Landing page
├── About-Me.html          # About section
├── Portfolio.html         # Project showcase
├── Contact-Me.html        # Contact information
├── style-Home.css         # Home page styling
├── style_About.css        # About page styling
├── style_Portfolio.css    # Portfolio page styling
├── style_Contact-Me.css   # Contact page styling
├── logo.png               # Site logo (used in navbar)
└── README.md              # Project documentation
```

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom styling and responsive design
- **Font Awesome 6.7.2** - Icon library
- **Google Fonts** - Typography (Josefin Sans, Playwrite VN)

## Getting Started

### Prerequisites
No special software required. Simply open in any modern web browser.

### Installation

1. Clone the repository or download the project files
   ```bash
   git clone https://github.com/VaguePrateek/Portfolio-Demo.git
   ```

2. Navigate to the project directory
   ```bash
   cd Portfolio-Demo
   ```

3. Open `Home.html` in your web browser
   ```bash
   # On Windows
   start Home.html
   
   # On macOS
   open Home.html
   
   # On Linux
   xdg-open Home.html
   ```

## Pages

### Home (`Home.html`)
The landing page with an introduction and navigation to other sections. Welcomes visitors and sets the tone for the portfolio.

### About Me (`About-Me.html`)
Personal background, skills, experience, and professional information about Prateek Dewangan.

### Portfolio (`Portfolio.html`)
Showcase of projects, work samples, and professional achievements with descriptions and links.

### Contact Me (`Contact-Me.html`)
Contact information and links to connect via email or social media platforms.

## Navigation

The site features a sticky navigation bar with:
- **Logo** - Links back to Home.html
- **Menu Items** - About me, Portfolio, Contact links (hidden on mobile)
- **Social Icons** - Links to Instagram, LinkedIn, and GitHub
- **Mobile Menu** - Hamburger menu for responsive navigation

## Styling

Each page has its own stylesheet:
- `style-Home.css` - Home page styles
- `style_About.css` - About page styles
- `style_Portfolio.css` - Portfolio page styles
- `style_Contact-Me.css` - Contact page styles

The stylesheets use responsive design principles with media queries to ensure optimal viewing on mobile, tablet, and desktop devices.
